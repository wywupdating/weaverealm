## 六、风格规范

### 6.1 公众号规范

| 项目 | 规范 |
|------|------|
| **语气** | 像朋友聊天，亲切有温度，不是在写通知 |
| **结构** | 信息分层清晰：氛围引入→核心信息→行动引导 |
| **段落** | 每段2-5行，不要大段文字 |
| **emoji** | 适度使用（信息标注处用📅💰📍），不过度 |
| **尾部** | 每篇固定带组织简介+搜索关键词 |
| **转发福利** | 课程招募类推文可加"转发集赞送代金券" |
| **长度** | 课程招募800-1200字，舞会预告500-800字，种草文600-1000字 |

### 6.2 小红书规范

| 项目 | 规范 |
|------|------|
| **语气** | 更轻松、更个人化，像发朋友圈 |
| **长度** | 正文200-400字，短而有料 |
| **emoji** | 多用，每行1-2个 |
| **标签** | 8-12个，包含城市+品类+场景 |
| **互动** | 结尾必须有互动引导（评论/私信） |
| **封面** | 封面图文字要大、醒目、一句话传达核心 |

### 6.3 不做的事

- ❌ 不要硬卖（摇摆野兽思思原话："过多售卖可能会引起订阅者的厌烦"）
- ❌ 不要用"官方通知"语气
- ❌ 不要大段文字堆砌
- ❌ 不要忽略新人顾虑（每次都要有"零基础友好"的信号）
