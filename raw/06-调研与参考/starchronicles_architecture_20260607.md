# 星域纪事 — 小说宇宙架构（项目快照 2026-06-07）

> 定位：用虚构世界短篇承载AI原生组织理论，GitHub开放共创

## 核心架构

**双线**：暗线（主角框架，后台维护）+ 明线（NPC视角短篇，前台发布）

**三层架构**（核心设计）：
- **核心层**：AI原生组织理论/架构/案例 — 100%真实可验证，2人approve
- **中间层**：知识短篇/理论解读 — 核心准确允许润色，1人approve
- **外层**：长篇小说/夸张短篇/同人 — 核心不崩即可，基本把关
- 内容可向上流动（外→中→核心），核心不降级

## 贡献模式

**三类贡献者**：理论贡献者（AI洞见）+ 叙事贡献者（节奏/人物/情感）+ 混合
- AI辅助流程：读资料→选方向→人类写理论→人类写映射→AI生成故事→提交PR
- 长篇开放生态：不限制谁写，不限制版本数，多个长篇=平行宇宙

## 冷启动（5阶段）
Phase 0：自己写5篇打样 → Phase 1：定向邀请程序员 → Phase 1.5：招募叙事贡献者 → Phase 2：开放共创 → Phase 3：长篇启动 → Phase 4：生态自驱
