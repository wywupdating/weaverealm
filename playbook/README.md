# 织境实操手册

> 给小组织的 AI 协作操作指南——不看理论，照着做就行。

## 这是什么

`playbook/` 是框架的可操作版本。如果你是一个小组织的运营者，不想啃理论，只想知道"第一步做什么、第二步做什么"，这里就是你的起点。

## 指南列表

> 🚧 实操指南持续补充中。以下为规划方向。

### 起步类
- **三步开张** — 你有一个小团队，想让AI帮你做事，从哪开始？
- **选对工具** — 市面上AI工具那么多，你的团队需要哪些？
- **第一个Agent** — 怎么把你的AI助手从"搜索引擎"调教成"靠谱搭档"

### 运转类
- **分配透明化** — 几个人一起干活，钱和时间怎么分才不吵架？
- **知识别带走** — 团队里有人要走，怎么确保他脑子里的东西留下来？
- **冲突降温** — 小团队吵架了，怎么快速降温继续干活？

### 成长类
- **诊断你的组织** — 你的团队健康吗？一份自检清单
- **找顾问** — 什么时候该找外部帮手，找什么样的？
- **第二曲线** — 业务增长停了，怎么找到新方向？

### 收尾类
- **体面散伙** — 团队做不下去了，怎么散才不留烂账？
- **知识开源** — 散伙前把经验整理出来，让别人少踩坑

## 使用方式

每份指南包含：
1. **适用场景**：什么情况下该看这份指南
2. **步骤清单**：按顺序做就行
3. **常见坑**：别人踩过的雷
4. **模板/工具**：可以直接套用的资源
