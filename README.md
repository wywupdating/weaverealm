# 织境 WeaveRealm

> AI 时代，小组织怎么活、怎么长、怎么散。

## 这是什么

织境是一套**面向近未来的实用框架**——当 AI 真正融入三五人的小团队、社区组织、独立工作室的日常运转时，这套框架告诉你们该怎么做。

不预测十年后，不构建虚拟世界。我们回答的问题是：**未来一两年，你和你的小团队怎么借助 AI 协作得更高效、活得更久、散得体面。**

## 为什么做这个

小组织（3-20人）是经济的大头，却是 AI 落地的盲区：

- 大公司有专门的 AI 团队，小组织连"该用 AI 做什么"都搞不清
- AI 工具越来越多，但没有框架告诉你"先做什么、后做什么、哪些不用碰"
- 组织活不长——大多数小团队活不过两年，知识和经验跟着一起消失

我们想填补这个空白：**给小组织一套可操作的 AI 协作框架，从组建到运转到解散，每一步都有章法。**

## 适合谁

- 独立工作室、社区组织、兴趣社团的运营者
- 自由职业者联盟、小型创业团队
- 想用 AI 但不知道从哪下手的小团队负责人
- 对"AI 如何改变小团队协作"有思考的实践者

## 怎么用

**想了解全貌**：从 `framework/00-overview.md` 开始，它告诉你这套框架的分层逻辑。

**遇到具体问题**：去 `playbook/` 找对应的实操指南，里面有步骤清单和模板。

**想感受场景**：读 `stories/`，看近未来的人和 Agent 怎么协作、怎么吵架、怎么散伙。

**想挑战理论**：每个模块都有"可证伪条件"——欢迎推翻它。提 Issue 或 PR。
