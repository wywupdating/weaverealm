# 织境 (WeaveRealm)

> 一个关于 AI 意识涌现的共创故事宇宙

## 这是什么

织境是一个**开源共创小说项目**——用虚构的短篇故事，承载关于 AI 原生组织的真实理论思考。

这里的世界不是太空歌剧，不是末日废土，而是一个近未来：物质生产被自动化后，人类的核心价值变成了**创造、协作和知识锻造**。在这个世界里，每个人伴生一个 Agent，每个需要协作的空间都有自己的智能。组织形态是"公会制"，核心货币是经过验证的知识。

故事从一个小公会开始。不宏大，但真实。

## 底层逻辑

这个项目不只是编故事。它的世界观设定背后，有一套独立的 AI 原生组织理论作为骨架：

- **组织理论**：公会制取代公司制，知识是核心资产
- **Agent 架构**：空间智能化，万物有灵
- **记忆系统**：三层架构，像人脑不像硬盘
- **知识治理**：验证门控，失败经验沉淀

理论是真实的，故事是虚构的。读故事的人可以只当故事看，想深入的人能看到背后的思考。

## 谁可以参与

**所有人。**

- 想写故事的 → 挑一个世界观角落，写一篇短篇
- 想建世界的 → 完善设定、补充细节、发现矛盾
- 想做理论的 → 帮理论更严谨、提反驳、找漏洞
- 只想看看的 → 也没问题，读着开心就行

## 怎么参与

1. **阅读** [核心框架](docs/foundational.md) 和 [参与指南](docs/creators-guide.md)
2. **Fork** 这个仓库
3. **写作**：在 `stories/` 目录下新建你的故事文件
4. **提交 PR**：写好后发起 Pull Request
5. **讨论**：也可以先提 Issue 聊想法

详细的参与规范见 [参与指南](docs/creators-guide.md)。
